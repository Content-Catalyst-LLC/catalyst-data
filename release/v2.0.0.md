# Catalyst Data v2.0.0

## Connected Evidence and Measurement Platform

Catalyst Data v2.0.0 consolidates the complete v1 evidence lifecycle into one connected, local-first platform. It preserves the stable `catalyst-data-record/1.0` contract while adding the `catalyst-data-platform/2.0` system manifest, contract registry, component registry, capability graph, immutable release snapshots, and cross-subsystem integrity verification.

### Added

- Migration 013 for connected-platform governance.
- Immutable platform contract registrations with SHA-256 schema checksums.
- Versioned platform components and capability declarations.
- Governed links for handoffs, APIs, data sources, analysis, publication, embeds, and federation.
- Immutable platform release snapshots.
- Append-only platform integrity checks and registry events.
- Integrated platform and operational readiness reports.
- Public v2 platform manifest and protected platform registry API surfaces.
- CLI workflows for component registration, contract synchronization, capability linking, snapshots, verification, integrity, and readiness.

### Compatibility

- All v1 record, evidence, review, query, API, connector, workspace, analysis, and operational contracts remain supported.
- Existing `/v1` HTTP endpoints remain available.
- Platform Core remains optional; local SQLite operation remains fully supported.
- Human review and approval remain required for externally published records.

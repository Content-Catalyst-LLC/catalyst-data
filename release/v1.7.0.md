# Catalyst Data v1.7.0 — Query, Comparison, and Export Studio

Catalyst Data v1.7.0 turns the governed repository into a reproducible local query and publication environment.

## Added

- `catalyst-data-query/1.0` query-definition contract.
- Migration 007 for saved queries, immutable definition versions, frozen runs, frozen record payloads, query warnings, and export bundle registrations.
- Filters across entity, indicator, period, source, framework, evidence, confidence, quality, review, publication, tags, and text.
- Consecutive-period comparisons using governed units and comparability rules.
- Explicit publication, quality, evidence, limited-comparison, and incompatible-comparison warnings.
- Reproducible query summaries and Markdown briefs.
- Deterministic ZIP or directory export bundles with records, CSV, comparisons, warnings, provenance, review history, data dictionary, and manifest checksums.
- CLI commands for saving, listing, executing, inspecting, briefing, and exporting queries.

## Compatibility

The canonical record contract remains `catalyst-data-record/1.0`. Migration 007 is additive and existing v1.6.0 repositories retain all records, review history, provenance, lineage, and governance data.

# Catalyst Data v1.11.0

## Analysis Artifacts and Reproducible Data Packages

Catalyst Data v1.11.0 connects governed records to analytical work without merging the repository with any one analysis product. It freezes exact inputs, parameters, environment details, code references, outputs, derived lineage, replication reviews, and deterministic package manifests.

### Included

- `catalyst-data-analysis-artifact/1.0` and packaged JSON Schema.
- Migration 011 with analysis artifacts, immutable versions and activations, runs, frozen input records, outputs, derived measurement lineage, platform links, replication reviews, invalidation events, resolutions, and package exports.
- Automatic upstream-record invalidation events when a frozen input changes.
- Deterministic reproducible ZIP and directory packages with manifests and SHA-256 checksums.
- Platform links for Workbench, Research Lab, Catalyst Analytics R, Decision Studio, Knowledge Library, Site Intelligence, Catalyst Canvas, and Platform Core.
- Analysis registration, execution-recording, inspection, packaging, invalidation, lineage, and replication CLI commands.

### Upgrade behavior

Existing v1.10.0 records, connectors, workspaces, API keys, queries, reviews, provenance, and access history are preserved. Migration 011 adds analytical history without changing `catalyst-data-record/1.0`. Populated migration 011 can be rolled back after its append-only triggers are removed; canonical data and migrations 001–010 remain intact.

### Release boundary

The release governs analytical artifacts and reproducibility evidence. It does not execute untrusted code, certify methods, or allow AI systems to approve results. Human reviewers remain responsible for interpretation, replication, and publication.

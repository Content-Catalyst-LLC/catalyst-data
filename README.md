# Catalyst Data

Catalyst Data is the shared SQL and evidence layer for Sustainable Catalyst. It connects entities, indicators, sources, periods, measurements, confidence levels, and method notes so sustainability and systems work can stay traceable over time.

The module is designed for auditable work: what was measured, where it came from, when it applied, how confident the record is, and what assumptions still need review.

## What this repo contains

- `schema.sql` — SQLite schema for entities, indicators, periods, sources, measurements, tags, and review notes.
- `queries.sql` — sample seed data and example review queries.
- `python/` — a small Catalyst Data brief generator and validation layer.
- `data/` — sample CSV inputs.
- `examples/` — sample project-level JSON input.
- `outputs/` — example JSON and Markdown outputs.
- `schemas/` — JSON schema for Catalyst Data exports.
- `wordpress/catalyst-data-demo/` — shortcode plugin source for the online demo.
- `dist/catalyst-data-demo.zip` — built WordPress plugin zip, generated by the upgrade script.
- `docs/` — methodology, schema, provenance, export, review, and WordPress plugin documentation.
- `tests/` — lightweight pytest coverage for the Python engine.

## WordPress demo

Install the plugin zip and add this shortcode to the Catalyst Data page:

```text
[catalyst_data_demo]
```

The demo is browser-based. It does not send records to a server. It creates a structured evidence record that can be copied or downloaded as JSON.

## Quick start: SQLite

```bash
sqlite3 catalyst_data.db < schema.sql
sqlite3 catalyst_data.db < queries.sql
```

Run useful review queries:

```bash
sqlite3 catalyst_data.db "SELECT * FROM measurement_review;"
sqlite3 catalyst_data.db "SELECT * FROM provenance_gaps;"
```

## Quick start: Python

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements-dev.txt
python python/generate_data_brief.py examples/sample_project.json outputs/generated_brief.md
pytest
```

```text
entity → indicator → period → measurement → source → confidence → review
```

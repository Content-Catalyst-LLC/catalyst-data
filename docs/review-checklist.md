# Review Checklist

Before treating a Catalyst Data record as reviewable, confirm:

- The entity and indicator are clearly identified.
- The reporting period is explicit.
- Baseline and current values use compatible units.
- A named source supports the measurement.
- Confidence is between 0 and 100.
- Method and assumption notes explain material limitations.
- `review_status` reflects evidence readiness rather than whether the value improved.
- `signal_status` reflects direction without implying evidence quality.
- Zero-baseline records are treated as indeterminate rather than as 0% change.
- The exported trace path is intact.
